## Chạy Linux cấu trúc x86_64 / amd64 trên Termux với giả lập QEMU + Proot!

### Câu lệnh:

```bash 
curl -O https://raw.githubusercontent.com/dragonx943/proot-qemu-termux/main/install.sh && chmod a+x install.sh && bash install.sh
```
